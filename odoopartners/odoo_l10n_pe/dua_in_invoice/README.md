# DUA in Invoice

## Summary

This module will create the field called "Year of issue" and "Customs unit" in the supplier invoice when we choose the document type "50" called "SAD".

In the field "Customs Unit" it will give us a list of codes: 019, 028, 046, 055, 082, etc.

*These codes will be loaded in the path "LOCATION/PLE/[11]CUSTOMS DEPENDENCY CODE". The file to be loaded is as follows [CODES](https://docs.google.com/spreadsheets/d/1Jb4izTYrdfFudL2Pc4LJSryj7-grnQImHOgZEqwzY4Q/edit#gid=0)
