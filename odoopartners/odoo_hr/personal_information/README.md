# Personal Information

## Summary

This module has been designed with the purpose of improving the management of employees' personal information by allowing the assignment of relatives.

In order to offer greater flexibility in data collection, the module introduces four new fields in the employee file:

- Names
- Last name
- Mother's last name
- Relatives

In addition to the addition of these custom fields, the module hides three native fields from the employee record that will not be used:

- Spouse's full name (spouse_complete_name)
- Spouse's date of birth (spouse_birthdate)
- Number of dependent children (children)

## Main Features

- Assignment of relatives to employees.
- Additional fields: First names, Paternal last name, Maternal last name.
- Hides native fields on the employee record that will not be used.
