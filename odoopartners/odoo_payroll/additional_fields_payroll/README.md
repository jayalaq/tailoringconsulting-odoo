# Additional Fields Payroll

## Summary

Allows you to create in the employee / contract module the icons for the reason for leaving the type of contract and remuneration in kind.

Create the fields for "Reason for Leaving", "Contract Type" and "In-kind Remuneration" in the pension system and enable the creation of icons for "Reason for leaving", "Contract type" and "In-kind Remuneration" in the employee/contract module.
