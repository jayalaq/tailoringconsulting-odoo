Peruvian Localization Base for EDI documents

l10n_pe_delivery_note
-
Add validation services for electronic invoicing, which allows you to validate directly with SUNAT, free of charge and in an unlimited way, or with any
authorized OSE provider. 

l10n_pe_edocument
-
Add validation services for electronic invoicing, which allows you to validate directly with SUNAT, free of charge and in an unlimited way, or with any
authorized OSE provider.

l10n_pe_edocument_rectifying
-
The electronic invoicing of the Peruvian location is extended, so that it allows issuing Credit Notes and Debt Notes, including for electronic invoices 
that have not been issued via Odoo.

l10n_pe_efact_decimal_precision
-
This module will allow us to force the label of the sale price of the product excluding taxes with more than several digits

qr_code_on_sale_invoice
-
This module will create the QR code for printing the invoice in digital format

qr_code_stock_picking
-
Generate the QR code in the printing of transfers in inventory

query_cdr_invoice
-
This module allows the consultation of the CDR in SUNAT and changes the status of the EDI Document to "Sent"
